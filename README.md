Hospital Management System – Zoho Creator
Project Overview

The Hospital Management System is a cloud-based application developed using Zoho Creator. The system is designed to manage and streamline hospital operations including patient management, doctor management, appointments, treatment planning, billing, and reporting.

This application demonstrates the use of Zoho Creator forms, reports, workflows, and data management features in building a structured hospital management solution.

Features

Patient Registration and Management

Doctor Management

Appointment Scheduling

Emergency Case Handling

Treatment Plan Management

Weekly Treatment Fee Report

Data Organization using Reports

Exportable Application (.ds file)

Weekly Treatment Fee Report

The system includes a report that calculates and displays weekly treatment fees collected. This report helps analyze revenue trends and monitor financial performance.
